# Intelligent CU (ICU)

An AI-powered intelligent assistant system for CUHK Chung Chi College, developed as a Senior Seminar project.

## 🌟 Overview

Intelligent CU is a campus-oriented AI assistant that leverages Large Language Models to provide intelligent support for CUHK Chung Chi College students and faculty.

## 📂 Repository

**GitHub:** [https://github.com/normalman743/fypv2](https://github.com/normalman743/fypv2)


## ✨ Key Features

### 🤖 AI-Powered Chat Assistant
- **Multi-Channel Knowledge Integration**: Combines CUHK-specific institutional knowledge with general academic resources
- **Real-time Streaming Responses**: Continuous conversation with stream output for natural interactions
- **RAG-Enhanced Accuracy**: Retrieval-Augmented Generation ensures contextually relevant and accurate answers
- **Online Search Integration**: Access to real-time web information for up-to-date answers beyond the knowledge base

### 📚 Smart Knowledge Management
- **CUHK-Specific Knowledge Base**: Pre-loaded with campus-specific information, policies, and resources
- **Personal Learning Materials**: Store and organize study materials in a student-friendly way
- **Context-Aware Referencing**: Automatically provides relevant references to LLM when needed

### 🎓 Academic Support
- Course and semester information queries
- Document and file management
- Academic resource navigation

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript
- **Backend:** FastAPI, Python
- **AI/ML:** GPTS with RAG
- **Database:** MYSQL
- **Vector Store:** ChromaDB
