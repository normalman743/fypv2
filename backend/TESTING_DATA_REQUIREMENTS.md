## 🛠️ 数据初始化脚本使用

### 🚀 快速初始化

```bash
# 初始化完整测试数据
python scripts/init_test_data.py init

# 检查数据状态
python scripts/init_test_data.py check

# 清理测试数据  
python scripts/init_test_data.py clean
```

### 📋 初始化内容

执行 `init` 命令将创建：

1. **1个管理员用户** (admin/admin123)
2. **4个测试用户** (testuser等/testpass123)  
3. **10+个有效邀请码** (TEST2025, INVITE2025等)
4. **3个测试学期** (包含当前和测试学期)
5. **10+个测试课程** (跨用户和学期)
6. **30+个文件夹** (覆盖所有类型)
7. **6+个测试聊天** (通用和课程聊天)
8. **20+条测试消息** (完整对话历史)

### 🔍 数据验证
